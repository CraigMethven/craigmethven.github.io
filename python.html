Boop 'em

Description

Game I made in advanced higher computing. The game involves taking it in turns to place counters on a grid. The players can either play in a blank spot or a spot they own. If the number of counters in a square equals the number of surrounding squares then the number of counter in the square gets spread to the surrounding squares and that square is taken by that player. The aim of the game is to eliminate the opponant.
